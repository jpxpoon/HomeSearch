HomeSearch App

How to run application:
1. Open .\HomeSearch\bin\Release\netcoreapp3.1\publish\HomeSearch.exe
2. Open Link in browser: https://localhost:5001

Usage:
- [Button]"Import Homes": Import csv file ("redfin.csv")
  * Data doesn't stack, new file will overwrite old data
- Table shows home's informations
- Use filters to search homes

Tech Stacks:
Frontend: HTML/CSS/JS, jQuery, vuejs, bootstrap
Backend: C#/.NET Core
Database: Sqlite
